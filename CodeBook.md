# Code Book
## Initial Data
The data is initially from the "Human Activity Recognition Using Smartphones" DataSet from "http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones".

## Initial Data to Clean Data 
**(see "run_analysis.R" script)**

1. read the txt-files for training and test Datasets
2. read the features.txt file
3. rename the test and training data-names according to the features
4. find the features for mean or standard deviations
5. select only the columns in the combined.data that are mean or standard deviation features
6. read the activity labels 
7. add the subject ids to the datasets
8. add the activity labels to the datasets
9. combine the training and test data
10. clean the variable names
11. calculate the mean of the variables by activity and subjectId
